Introduction

The charmap is the list of supported characters with the respective numeric codes and the address relating to the font to be loaded from the dedicated memory area to be printed on screen. The single font is a bitmap with the resolution of 5 x 8 px (40 bits)
ANC16 Standard Operative Systems follow the following charmap:
